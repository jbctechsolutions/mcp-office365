Get Started

Up and running in minutes

Add the MCP server to your preferred AI client and start managing Microsoft 365.

1

Install

No global install required. Run directly with npx:

bash 
npx -y @jbctechsolutions/mcp-office365
2

Configure Claude Desktop

Add this to your Claude Desktop configuration file:

json 
{
  "mcpServers": {
    "office365": {
      "command": "npx",
      "args": ["-y", "@jbctechsolutions/mcp-office365"]
    }
  }
}
3

Or use with Claude Code

Add the server directly from the command line:

bash 
claude mcp add office365 -- npx -y @jbctechsolutions/mcp-office365

Safety

Two-phase confirmation

Destructive operations like sending, deleting, or modifying data use a prepare/confirm pattern to keep you in control.

Phase 1

Prepare

The AI agent calls a prepare tool (e.g. mail_send_prepare) which validates the request and returns a preview with a confirmation token. Nothing is executed yet.

Phase 2

Confirm

The agent reviews the preview with you. Only when you approve does it call the confirm tool with the token, executing the action. Tokens expire to prevent stale operations.

Why this matters: AI agents can misinterpret intent. The two-phase pattern ensures a human always reviews destructive actions before they happen, without slowing down read-only operations.
